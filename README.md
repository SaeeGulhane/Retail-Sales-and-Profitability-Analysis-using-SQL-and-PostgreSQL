# Retail Sales and Profitability Analysis using SQL and PostgreSQL

# Project Overview

This project analyzes retail sales performance and profitability trends using structured SQL queries on a PostgreSQL database. It aims to extract valuable business insights from sales data such as city-wise revenue, product category performance, customer behavior, and profitability distribution across branches.

The project demonstrates SQL proficiency in data analysis, aggregation, and business intelligence, turning raw transactional data into actionable insights.

# Skills Demonstrated

* SQL Querying and Optimization
* Data Aggregation and Grouping
* Joins and Subqueries
* Window Functions
* Business and Financial Analysis
* PostgreSQL Database Management

# Results and Insights
1. Sales Insights
    * Identified key product categories and branches driving the highest sales.
    * Determined the most preferred payment methods among customers.
2. Profitability
    * Highlighted the most profitable product categories and locations.
    * Analyzed overall revenue distribution across branches and cities.
3. Customer Behavior
    * Observed trends in customer ratings and payment preferences.
    * Identified peak shopping hours and high-activity regions.
